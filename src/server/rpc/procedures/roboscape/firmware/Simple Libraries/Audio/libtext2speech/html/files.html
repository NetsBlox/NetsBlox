text2speech.hPronounces words spelled out phonemically. This library is a spin2cpp.exe translation of Phil Pilgrim's Phonemic voice synthesizer library. Phil's library builds on Chip Gracey's VocalTract module to produce phonemic-based speech. The formant values it uses are based heavily on work done by D.H. Klatt in the "KLSYN" formant synthesis program
